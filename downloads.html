File Package

The following File Package contains an NPM file, which is used by various FHIR tools. This file encompasses all value sets, profiles, extensions, list of pages, and URLs in the Implementation Guide, among others, defined in this version of the Implementation Guides. When it comes to generating implementation artifacts, this file should be prioritized as it contains the rules that determine the validity of the profiles. Implementers should also be familiar with the content of the specification and the relevant profiles to ensure a compliant implementation. For more details, see the documentation on validating profiles and resources in FHIR.
