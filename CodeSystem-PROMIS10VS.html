CodeSystem: PROMIS-10 Response Questionnaire

Official URL: https://molic-avc.gabriellesantosleandro.com/CodeSystem/PROMIS10VS Version: 0.0.0
Active as of 2024-01-11 Computable Name: PROMIS10CS

CodeSystem that defines the response values for the PROMIS-10 (Patient-Reported Outcomes Measurement Information System 10-item short form) Questionnaire.

This Code system is referenced in the content logical definition of the following value sets:

  • PROMIS10VS
